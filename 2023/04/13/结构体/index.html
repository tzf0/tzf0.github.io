结构体

结构体的定义和使用

定义

struct Student
{
string name;
int age;
int score;
};

使用

struct Student s; s.name = "张三"; s.age = 18; s.score = 100;

struct Student s = { "张三", 18, 100 };

Student s = { "张三", 18, 100 };

结构体数组

将自定义的结构体放到数组中方便维护

struct 结构体名 数组名[元素个数] = { {}， {}，... {}};

结构体指针

定义

struct Student* p = &s;

Student* p = &s;

通过指针访问

cout << "姓名：" << p->name << endl;

姓名：张三

结构体中const的使用

传递方式：值传递，拷贝时会拷贝数据（尽量用地址传递）
