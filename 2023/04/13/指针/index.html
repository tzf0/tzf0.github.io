指针

解引用

可以通过解引用的方式来找到指针指向的内存

指针前面加 * 代表解引用，找到指针指向的内存中的数据

cout << "a的地址为：" << &a << endl; cout << "*p的值为：" << *p << endl;

a的地址为：10
*p的值为：10

指针所占的内存空间

在32位操作系统下，占4个字节

在64位操作系统下，占8个字节

int a = 10; int* p = &a;

sizeof (int) = 8
sizeof (int) = 8**

空指针

空指针用于给指针变量初始化

空指针不可以访问

空指针指向内存为0的空间

0~255之间的内存编号是系统占用，不可以访问

int* p = NULL; cout << *p << endl;

(进程 25944)已退出，代码为 -1073741819。

野指针

指针变量指向非法的内存空间

int* p = (int*)0x1100; cout << *p << endl;

(进程 3160)已退出，代码为 -1073741819。

const 修饰指针

1.const修饰指针 ——常量指针

​ 指针的指向可以修改，但是指针的值不可以改

​ int a = 10;

​ int b = 10;

​ const int* p = &a;

​ p = &b;

2.const修饰常量 ——指针常量

​ 指针的指向不可以修改，但是指针的值可以改

​ int a = 10;

​ int b = 10;

​ int* const p = &a;

​ p = 20;

3.const即修饰指针，又修饰常量

​ 指针的指向和指针指向的值都不能改

​ int a = 10;

​ int b = 10;

​ const int* const p = &a;

​ p = 20;
